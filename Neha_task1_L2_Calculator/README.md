# Calculator Web Application

## About the Project

This project is a simple calculator created using HTML, CSS, and JavaScript. The calculator can perform basic mathematical operations like addition, subtraction, multiplication, and division.

The main aim of this project is to understand how a web application works and how different web technologies are used together to create an interactive calculator.

---

## Technologies Used

* HTML
* CSS
* JavaScript
* vs code

---

## How the Project Works

* HTML is used to create the structure of the calculator like buttons and display screen.
* CSS is used to design the calculator and make it look attractive and properly aligned.
* JavaScript is used to make the calculator functional and perform calculations when buttons are clicked.

The calculator takes input from the user through buttons and displays the result on the screen.

---

## Features

* Addition, subtraction, multiplication, and division operations
* Clear button to remove all values
* Delete button to remove the last entered value
* Responsive and simple user interface
* Easy to use design
