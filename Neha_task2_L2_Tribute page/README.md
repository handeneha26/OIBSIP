# Tribute Page – Ratan Tata

This project is a responsive Tribute Page created using HTML and CSS.
The webpage is dedicated to Ratan Tata, one of India’s most respected industrialists and philanthropists.

The page includes:

* A modern and attractive user interface
* A responsive layout that fits different screen sizes
* An image section with Ratan Tata’s photograph
* A quote section
* A short biography section
* Stylish background, colors, and typography

## Technologies Used

* HTML5
* CSS3
* vs code

## Features

* Fully responsive design
* Clean and modern layout
* Centered content structure
* Mobile-friendly interface
* Smooth styling using CSS

## What I Learned

Through this project, I learned:

* How to structure a webpage using HTML
* How to style elements using CSS
* How to create responsive web designs
* How to align content properly using Flexbox
* How to work with images, fonts, spacing, and layouts

## Conclusion

This project helped improve my front-end web development skills and understanding of responsive webpage design using HTML and CSS.
