#  To-Do List Web App

##  Project Overview

The To-Do List Web App is a simple and interactive task management application developed using HTML, CSS, and JavaScript. This project helps users organize their daily activities by adding, managing, and tracking tasks efficiently.

The application provides a clean and user-friendly interface where users can create tasks along with descriptions and manage them easily.

---

##  Features

* Add new tasks with title and description
* View all pending tasks
* Mark tasks as completed
* Separate section for completed tasks
* Edit existing tasks
* Delete tasks
* Displays date and time when a task is added
* Displays completion time for completed tasks
* Fully responsive design for desktop and mobile devices

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript

---

##  Working of the Project

The user enters a task title and description in the input fields and clicks on the **Add Task** button. The task is then displayed in the **Pending Tasks** section along with the current date and time.

Users can:

* Mark a task as complete
* Edit task details
* Delete tasks whenever needed

Once a task is completed, it automatically moves to the **Completed Tasks** section along with the completion time.

The project uses JavaScript DOM manipulation to dynamically create, update, and remove tasks without refreshing the page.
