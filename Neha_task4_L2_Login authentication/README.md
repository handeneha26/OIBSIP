# Authentication System using C++

## Overview

This project is a simple Authentication System developed using C++. The main purpose of this project is to understand the basic concept of user authentication including registration, login validation, and secured access.

The system allows users to create a new account, login using their credentials, and access a secured page after successful authentication. User data is stored in a text file which acts as a small database for this project.

## Features

* User Registration
* User Login Authentication
* Secure Access Page
* File Handling using C++
* Simple and User-Friendly Console Interface

## How the Project Works

The project starts with a menu that provides three options:

1. Register
2. Login
3. Exit

### Registration

When the user selects the registration option, the program asks for a username and password. The entered details are then stored in a text file named `users.txt`.

### Login

In the login section, the user enters the registered username and password. The program checks the entered details with the stored records in the file.

### Secured Page

If the login credentials are correct, the user gets access to a secured dashboard page. If the details are incorrect, an error message is displayed.
